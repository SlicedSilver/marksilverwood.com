Research Project:
Application Of Stochastic Programming Techniques To Airline Scheduling

Read Abstract
Close Abstract

The goal of this project was to evaluate the effectiveness of stochastic programming techniques when applied to the airline scheduling process to reduce the effect of stochastic flight delays. A variety of traditional and stochastic programming models were developed for generating flight schedules. The resultant flight schedules were tested using simulations to evaluate their performance in real-world conditions with regard to flight delays, and their effects on the schedule’s operations. It was found that stochastic programming techniques were able to improve the delay recovery performance of the schedules at the cost of decreasing the schedule’s profit; and that flight schedules which are more dense with flight activity are affected more by the stochastic programming techniques. The use of stochastic programming techniques is recommended for the cases where an airline’s flight schedule has a high density of activity and the negative effects of flight delays needs to be minimized.

Honours Research Project:
Item Shortages: Data Mining Diagnostics

Read Abstract
Close Abstract

The UCW partnership is a significant manufacturer of rolling stock and the dominant player in the South African railway industry. UCW has been experiencing large parts shortages. An ERP system has recorded large amounts of data regarding these shortages. The project requires the analysis of the recorded data set using data mining techniques. It was found that only approximately 5% of the shortage report represented late parts which were required. The majority of the shortage report contained parts which still required ordering from the suppliers. It was determined that traditional data mining techniques were not appropriate for this specific application and that exploratory analysis techniques were more suitable for this specific case. The data mining techniques were suspected to have performed poorly due to the amount and selection of independent variables available in the data set. A restructuring and further splitting of the shortage report was recommended as well as the creation of more comprehensive data warehouse to allow more meaningful results to be obtained using data mining techniques in the future.

Honours Design Project:
Charter Airline Scheduling Model

Read Abstract
Close Abstract

Tourist charter airlines in Southern Africa have unique scheduling problems relating to their operational environment. Due to daily varying passenger demand and constraints, a new flight schedule is required to be formulated daily. The objective of the project was to design a flexible scheduling model for a charter airline using constraint programming principles. The scheduling model was required to obtain near-optimal solutions within a reasonable amount of time. A constraint integer program (CIP) model was developed and applied to a real-life charter aircraft scheduling problem. The CIP model was found to generate superior schedules, with respect to cost, compared with the existing baseline technique and cost savings of 12.1% were realised. The cost savings resulted from the CIP model requiring fewer total flights to satisfy the passenger demand. Further testing of the model using a wider spectrum of conditions is recommended for future work along with additional optimisation of the CIP model to decrease the required processing time.
